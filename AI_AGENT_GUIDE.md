## Setup Instructions

### Step 1: Get a Gemini API Key

1. Visit [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy the generated API key

> **Note**: The Gemini API has a generous free tier that should be sufficient for personal use.

### Step 2: Configure the Application

1. Open the `.env` file in your project root directory
2. Find the line: `GEMINI_API_KEY=your_gemini_api_key_here`
3. Replace `your_gemini_api_key_here` with your actual API key
4. Save the file

### Step 3: Restart the Server

If the server is running, restart it to load the new API key:

```bash
# Stop the current server (Ctrl+C)
# Then restart
.\run_app.ps1
```

## Troubleshooting

### Agent Not Responding

**Problem**: Chat button appears but agent doesn't respond

**Solutions**:

1. Check that `GEMINI_API_KEY` is set in `.env`
2. Verify the API key is valid
3. Restart the server
4. Check browser console for errors (F12)

### "API Key Not Set" Error

**Problem**: Error message about missing API key

**Solution**:

1. Ensure you've added the API key to `.env`
2. Restart the application server
3. Clear browser cache and reload

### Network Errors

**Problem**: "Network error" or "Failed to get response"

**Solutions**:

1. Check your internet connection
2. Verify the backend server is running
3. Check if you're logged in (token valid)
4. Try refreshing the page

### Rate Limiting

**Problem**: "Too many requests" error

**Solution**:

- Gemini free tier has rate limits
- Wait a few minutes before trying again
- Consider upgrading to a paid tier if needed
